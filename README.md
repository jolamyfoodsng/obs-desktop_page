# OBS Plugin Installer

OBS Plugin Installer is a simple desktop app that helps you install and manage OBS Studio plugins safely — without dealing with ZIP files, folders, or confusing install steps.

If you’ve ever downloaded a plugin and wondered *“where do I put this?”*, this app solves that.

---

## 🚀 What this app does

OBS Plugin Installer helps you:

- Discover useful OBS plugins
- Install plugins in one click
- Avoid breaking OBS with incorrect installs
- See which plugins are installed
- Get notified when updates are available

Everything is handled for you — no manual extraction, no guessing paths.

---

## 🧠 Why this exists

Installing OBS plugins manually can be frustrating:

- You download a ZIP file
- You don’t know where OBS is installed
- You’re unsure which folder to copy files into
- One mistake can break your setup

OBS Plugin Installer removes all of that.

---

## 📦 What’s included

- A curated list of tested OBS plugins
- Safe install process with validation
- Automatic OBS detection (no setup needed in most cases)
- Support for Windows, macOS, and Linux
- Built-in update system

## 💻 Download

To download the app:

1. Go to the **Releases** page on this repository
2. Download the correct file for your system

### Windows
- `.exe` → recommended (just run it)
- `.msi` → optional (for advanced / enterprise installs)

### macOS
- `.dmg`

### Linux
- `.AppImage` (recommended)
- `.deb` (for Debian/Ubuntu)
